# HR Attrition Analysis Dashboard

## 📊 Project Overview
This project analyzes employee attrition trends using Excel and Power BI.

## 🛠 Tools Used
- Power BI
- Excel

## 📁 Files Included
- Excel (Raw Data + MIS Report)
- Power BI Dashboard (.pbix)
- Dashboard Screenshots

## 📌 Key Metrics
- Total Employees
- Attrition Rate
- Active Employees

## 🔍 Analysis
- Attrition by Department
- Attrition by Job Role
- Overtime vs Attrition
- Salary impact on attrition

## 💡 Key Insights
- Overtime employees have higher attrition
- Lower salary employees tend to leave more
- Certain roles show higher turnover
